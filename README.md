# Mobile-Detection-System
🚀 Overview  An embedded IoT system designed to detect and count nearby electronic devices using RF, Wi-Fi, and Bluetooth signals. The system monitors surrounding device activity in real time and logs the data for analysis.

## ⚙️ Features
- Detects nearby devices using RF, Wi-Fi, and Bluetooth signals
- Counts number of devices in the surrounding area
- Real-time signal monitoring
- Data logging for analysis
- Multi-protocol detection in a single system

## 🛠 Tech Stack
- Embedded C / C++
- Microcontroller (Arduino / ESP8266)
- RF Module
- Wi-Fi Module
- Bluetooth Module

## 🔌 System Architecture
- RF, Wi-Fi, and Bluetooth modules scan for signals
- Microcontroller collects signal data
- Device presence is detected and counted
- Data is stored/logged for monitoring
  
## 🔄 How It Works
- System continuously scans for nearby signals
- RF/Wi-Fi/Bluetooth modules detect active devices
- Microcontroller processes incoming signals
- Devices are counted based on detection logic
- Data is stored for analysis
  
## 📊 Use Cases
- Crowd density monitoring
- Smart environment sensing
- Security and surveillance systems
- Network activity awareness
  
## ⚠️ Limitations
- Cannot uniquely identify individual users/devices
- Detection accuracy depends on signal strength
- Possible duplicate counting across protocols
- Limited range based on hardware
  
## 🚀 Future Improvements
- Dashboard for real-time visualization
- Filtering duplicate devices
- Integration with cloud database
- AI-based pattern detection
